# ethereum-smartcontract-snippet

# Requeriments
* Solidity compiler
* abigen tool (go-ethereum devtools)
* nodejs
* protoc (protobuf dev libs)
* golang 

```
brew install solidity
```
# install abigen:
```
$ go get -u github.com/ethereum/go-ethereum/...
$ cd $(go env GOPATH)/src/github.com/ethereum/go-ethereum
$ make
$ make devtools
```
